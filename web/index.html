Simple Installation

Follow these steps to get started with WaEnhancer.

  1. Ensure your device is Rooted.
  2. Install LSPosed (Zygisk or Riru).
  3. Download and install WA Enhancer APK.
  4. Enable the module in LSPosed Manager.
  5. Select WhatsApp in the scope.
  6. Force Stop WhatsApp and reopen.
